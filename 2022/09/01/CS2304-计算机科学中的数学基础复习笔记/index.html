集合论

势（Cardinality）

用于比较集合的大小
等势：$\mathbb{A}\approx \mathbb{A} \iff$ $A$和$B$存在双射函数

例：

  • $\mathbb{N} \times \mathbb{N} \approx \mathbb{N}$
  • $\mathbb{N} \approx \mathbb{Q}$
  • $(0,1) \approx \mathbb{R}$

Cantor Theorem：集合不与他的幂集等势

Proof.
Let $f:A\rightarrow P(A)$
Construct $B = {x|x\in A, x\notin f(x)}$
Then $B\in A$, but for each $x\in A$, $x\in B \iff x\notin f(x)$
Hence $B\notin f(x)$

序（Ordering）

$A\le B \iff$存在$A$到$B$的单射函数
Schröder-Bernstein Theorem：$A\le B \land B\le A \rightarrow A\approx B$

Proof.
Let $f:A\rightarrow B,g:B\rightarrow A$
Define $Cn$: $C_0 = A- ran(g), C{n+1}=g[f[C_n]]$
Thus $h(x)$ =
$\begin{cases}
& f(x) \text{ if } x\in C_n \text{ for some n}\
& g^{-1}(x)\text{ otherwise }
\end{cases}$
$h(x)$ is bijection.

组合（Combinatorics）

Twelvefold way

n 个不同的球，m 个不同的盒子

unrestricted

相当于 $f: N\rightarrow M$ 个数 $m^n$， 归纳法证明。

<= 1

相当于单射 $f:n\rightarrow M$ 个数 $(m)_n$，归纳法证明。

$(m)_n = m(m-1)(m-2)\dots (m-n+1)$
求 n 个元素的集合的置换数：$n!$
置换：有限集对自身的双射函数

>= 1

n 个相同的球，m 个不同的盒子

unrestricted

$x_1 + x_2 + \dots + x_m = n$ 的非负整数解数 $\binom{n+m-1}{m-1}$。
插板法证明。

<= 1

在 m 个盒子中选 n 个装球，总数为 $\binom{m}{n}$。

>= 1

$x_1 + x_2 + \dots + x_m = n$ 的正整数解数 $\binom{n-1}{m-1}$。
转化为插板法后证明。

$\binom{m}{n} = \binom{m}{m-n}$
$\binom{m-1}{n} + \binom{m-1}{n-1}= \binom{m}{n}$

n 个不同的球，m 个相同的盒子

unrestricted

<= 1

显然，当 $n\le m$ 时仅 1 种，反之则为 0 种。

>= 1

n 个相同的球，m 个相同的盒子

unrestricted

<= 1

显然，当 $n\le m$s 时仅一种，反之则为 0 种。

>= 1
